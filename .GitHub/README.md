<h1 align="center">в”ҖгҖҢ З«бҙңЙӘбҙ„бҙӢ sбҙҮбҙӣбҙңбҙҳ гҖҚв”Җ</h1>

 1. **рҹ”§ бҙңбҙҳбҙ…бҙҖбҙӣбҙҮ & бҙңбҙҳЙўКҖбҙҖбҙ…бҙҮ**
   ```bash
   sudo apt-get update && sudo apt-get upgrade -y
   ```

 2. **рҹ“Ұ ЙӘЙҙsбҙӣбҙҖКҹКҹ КҖбҙҮЗ«бҙңЙӘКҖбҙҮбҙ… бҙҳбҙҖбҙ„бҙӢбҙҖЙўбҙҮs**
   ```bash
   sudo apt-get install python3-pip ffmpeg -y
   ```
 4. **рҹ“Ң sбҙҮбҙӣбҙӣЙӘЙҙЙў бҙңбҙҳ бҙҳЙӘбҙҳ**
   ```bash
   sudo pip3 install -U pip
   ```
 5. **вҡЎ ЙӘЙҙsбҙӣбҙҖКҹКҹЙӘЙҙЙў ЙҙбҙҸбҙ…бҙҮ**
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash && source ~/.bashrc && nvm install v18
   ```
 6. **рҹ“Ҙ бҙ„КҹбҙҸЙҙбҙҮ бҙӣКңбҙҮ КҖбҙҮбҙҳбҙҸsЙӘбҙӣбҙҸКҖКҸ**
   ```bash
   git clone https://github.com/Im-Notcoder/Shivi-V2 && cd Shivi-V2
   ```
 7. **рҹ“Ӯ ЙӘЙҙsбҙӣбҙҖКҹКҹ КҖбҙҮЗ«бҙңЙӘКҖбҙҮбҙҚбҙҮЙҙбҙӣs**
   ```bash
   pip3 install -U -r requirements.txt
   ```
 8. **рҹ“қ бҙ„КҖбҙҮбҙҖбҙӣбҙҮ .env бҙЎЙӘбҙӣКң sбҙҖбҙҚбҙҳКҹбҙҮ.env**
   ```bash
   cp sample.env .env
   ```
   - Edit .env with your vars
 9. **вңҸпёҸ бҙҮбҙ…ЙӘбҙӣЙӘЙҙЙў бҙ бҙҖКҖs**
   ```bash
   vi .env
   ```
   - Edit .env with your values.
   - Press `I` button on keyboard to start editing.
   - Press `Ctrl + C`  once you are done with editing vars and type `:wq` to save .env or `:qa` to exit editing.
10. **рҹ”— ЙӘЙҙsбҙӣбҙҖКҹКҹЙӘЙҙЙў бҙӣбҙҚбҙңx**
    ```bash
    sudo apt install tmux -y && tmux
    ```
11. **рҹҡҖ КҖбҙңЙҙ бҙӣКңбҙҮ КҷбҙҸбҙӣ**
    ```bash
    bash start
    ```
